# BLE_Scan
Scan all nearby BLE device in terminal for OSX (written in swift)


# Install and Run
    - Download BLE_scan from the latest release
    - open terminal
    - find the folder of the downloaded BLE_scan
    - do a chmod +x ./BLE_scan
    - execute BLE_scan
    - ./BLE_scan
    - Enjoy
    - It Ctrl + C (when done)
